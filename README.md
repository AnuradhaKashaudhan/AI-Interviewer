# Multimodal AI Interviewer

This repository contains the codebase for the Multimodal AI Interviewer project.

## Project Structure

- `backend/`: Contains the backend code.
- `frontend/`: Contains the frontend UI application.
- `data/resumes/`: Directory for storing candidate resumes (ignored by git).
- `data/recordings/`: Directory for storing audio/video recordings (ignored by git).
- `models/`: Directory for storing downloaded or trained AI models (ignored by git).
