# 🚀 AI Interviewer & ATS Optimization Platform

An end-to-end AI-powered platform designed to help candidates **analyze resumes, improve ATS scores, and simulate real interview experiences using AI, Computer Vision, and NLP**.

---

## 🌟 Overview

This project provides a complete ecosystem for interview preparation by combining:

- 📄 Resume (CV) analysis
- 🤖 AI-driven mock interviews
- 🎥 Real-time video/audio recording
- 📊 Performance evaluation & feedback

---

## ✨ Key Features

### 📄 ATS Resume Analyzer
- Upload resume (PDF/Image)
- ATS-based scoring system
- Keyword optimization suggestions

### 🎤 AI Mock Interview
- Technical / Behavioral / Mixed interviews
- Questions generated from CV
- Real interview simulation

### 🎥 Live Interview + Recording
- Camera & mic activation (OpenCV)
- Full interview recording

### 📊 Performance Feedback
- Communication analysis
- Confidence & technical evaluation

### 📥 Download Recording
- Save interview video for improvement

---

## 🧠 Tech Stack

- **Backend:** Python (Flask/FastAPI)  
- **AI/ML:** NLP, ML Models  
- **Computer Vision:** OpenCV  
- **Frontend:** HTML, CSS, JavaScript  
- **Tools:** Git, APIs  

---

User → Upload CV → AI Processing → Question Generator
↓
Interview Module → Camera/Mic → Recording
↓
AI Evaluation → Feedback Report
